This article contains code and resources for the Iris Flower Classification project. The objective of this project is to classify iris flowers into distinct species based on their sepal and petal measurements. The dataset used for training and evaluation is the well-known Iris dataset, consisting of samples from three iris species: Setosa, Versicolor, and Virginia.
